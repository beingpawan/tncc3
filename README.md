# Technical & Coding Club (TnCC)

> Official repository of the Technical & Coding Club (TnCC).

---

## About the Club

The Technical & Coding Club is a collaborative platform where students develop technical skills, build projects

We aim to 
- Strong programming fundamentals
- Practical project-building experience
- Open-source contribution culture
- Peer-to-peer learning

---

## Mission

To create a technically empowered student community capable of solving real-world problems through code and innovation.

---

## Activities

- Hackathon participation
- Project-based learning

## Repository Purpose

This repository is used to:

- Share learning resources
- Archive workshop materials
